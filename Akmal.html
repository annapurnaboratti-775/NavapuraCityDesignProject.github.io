Implementation Steps (ADS)

  • Model all junctions, hospitals, fire & police stations as nodes in a weighted graph.
  • Store live status of each vehicle (free/busy, location) in hash tables.
  • Use priority queues to decide which vehicle should be dispatched first.
  • Run graph traversals to ensure all critical regions are reachable from at least one station.

Algorithms / Data Structures Used

Graph (Adjacency List) Hash Table Priority Queue (Heap) BFS / DFS

Impact: Enables unified monitoring of ambulances, fire trucks and patrol units, ensuring that the nearest available resource is dispatched with minimal delay.
