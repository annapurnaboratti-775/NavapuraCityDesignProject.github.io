Navapura City Initiatives

1. Urban Waste-to-Resource Hubs

SDG 11.6

Implementation Steps (ADS)

  • Represent waste collection points and processing hubs as graph nodes; transportation routes are edges.
  • Apply Dijkstra’s Algorithm to determine optimal routes for transporting waste to hubs.
  • Use Heaps / Priority Queues to prioritize high-impact or hazardous waste first.
  • Store historical waste generation data in Lookup Tables for quick retrieval.

Algorithms / Data Structures Used

  • Dijkstra’s Algorithm
  • Heap / Priority Queue
  • Lookup Table
Learn Dijkstra Learn Heap / PQ Learn Lookup Table

2. Neighborhood Water Resilience Centers

SDG 6

Implementation Steps (ADS)

  • Track water usage per household using Segment Trees / Fenwick Trees for efficient range queries.
  • Model water distribution as a graph and use BFS to check connectivity and detect leaks.
  • Store household and facility data in Binary Search Trees for fast access.

Algorithms / Data Structures Used

  • BFS
  • Segment Tree / Fenwick Tree
  • Binary Search Tree (BST)
Learn BFS Learn Segment/Fenwick Tree Learn BST
